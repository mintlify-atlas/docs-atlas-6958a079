## Overview

The YouTube API service (`src/services/youtube-api.js`) provides comprehensive access to music metadata, search functionality, and content discovery. All functions are exported for use throughout the application.

**Base URL**: `${YOUTUBE_API}` (resolves to `${API_URL}/api/youtube`)

**Authentication**: All requests include Bearer token in headers

```javascript
const HEADERS = { Authorization: `Bearer ${import.meta.env.VITE_YOUTUBE_API_TOKEN}` };
```

## Search Functions

### searchTracks

Search for music tracks with optional pagination.

<ParamField path="query" type="string" required>
  Search query string
</ParamField>

**Returns**: `Promise<{ tracks: Track[], continuation: string | undefined }>`

<CodeGroup>
```javascript Function Call
import { searchTracks } from '@/services/youtube-api';

const result = await searchTracks('bohemian rhapsody');
console.log(result.tracks); // Array of track objects
console.log(result.continuation); // Token for next page
```

```javascript Implementation
export async function searchTracks(query) {
  try {
    const res = await axios.get(`${YOUTUBE_API}/search`, { 
      params: { q: query, filter: 'SONG' }, 
      headers: HEADERS 
    });
    return { 
      tracks: (res.data.data?.items || []).map(mapSong), 
      continuation: res.data.data?.continuation 
    };
  } catch (e) { 
    return { tracks: [] }; 
  }
}
```
</CodeGroup>

**Endpoint**: `GET ${YOUTUBE_API}/search?q={query}&filter=SONG`

<ResponseField name="tracks" type="Track[]">
  Array of track objects with normalized structure
</ResponseField>
<ResponseField name="continuation" type="string">
  Pagination token for loading more results
</ResponseField>

## Discovery Functions

### getSuggestions

Get autocomplete suggestions for search queries.

<ParamField path="query" type="string" required>
  Partial search query
</ParamField>

**Returns**: `Promise<string[]>`

<CodeGroup>
```javascript Usage
const suggestions = await getSuggestions('beatl');
console.log(suggestions);
// ['beatles', 'beatles songs', 'beatles abbey road', ...]
```

```javascript Implementation
export async function getSuggestions(query) {
  try {
    const res = await axios.get(`${YOUTUBE_API}/search/suggestions`, { 
      params: { q: query }, 
      headers: HEADERS 
    });
    return res.data.data?.queries || [];
  } catch (e) { 
    return []; 
  }
}
```
</CodeGroup>

**Endpoint**: `GET ${YOUTUBE_API}/search/suggestions?q={query}`

**Source**: `src/services/youtube-api.js:168`

## Data Transformation

The service includes internal mapper functions that normalize API responses:

<Expandable title="mapSong (Track Mapper)">
```javascript
const mapSong = (item) => ({
  trackId: item.videoId || item.id || item.browseId,
  title: item.title,
  artists: item.artists || [],
  album: item.album 
    ? { title: item.album.name, albumId: item.album.id || item.album.browseId } 
    : null,
  thumbnailUrl: item.thumbnail,
  duration: { label: item.duration || "0:00" },
  isExplicit: item.explicit || false
});
```
Transforms raw YouTube Music track data into standardized Track objects.
</Expandable>

<Expandable title="mapAlbum (Album Mapper)">
```javascript
const mapAlbum = (item) => ({
  albumId: item.browseId || item.id,
  title: item.title,
  year: item.year || '',
  isExplicit: item.explicit || false,
  artist: item.artists?.[0] || { name: 'Unknown Artist' },
  thumbnailUrl: item.thumbnail
});
```
Transforms raw album data into standardized Album objects.
</Expandable>

<Expandable title="mapArtist (Artist Mapper)">
```javascript
const mapArtist = (item) => ({
  artistId: item.browseId || item.id,
  name: item.title || item.name || '',
  subscribers: item.subscribers || '',
  thumbnailUrl: item.thumbnail
});
```
Transforms raw artist data into standardized Artist objects.
</Expandable>

<Expandable title="mapPlaylist (Playlist Mapper)">
```javascript
const mapPlaylist = (item) => ({
  playlistId: item.browseId || item.id,
  title: item.title,
  itemCount: item.songCountText || '',
  author: item.author ? [item.author] : [],
  thumbnailUrl: item.thumbnail
});
```
Transforms raw playlist data into standardized Playlist objects.
</Expandable>

## Error Handling

All functions implement consistent error handling:

- Network errors return empty results (arrays or objects)
- No exceptions propagated to calling code
- Console logging in development mode
- Graceful degradation ensures UI stability

```javascript
try {
  const res = await axios.get(endpoint, { headers: HEADERS });
  return res.data.data || defaultValue;
} catch (e) {
  console.error('Error:', e);
  return defaultValue;
}
```
